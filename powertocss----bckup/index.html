Quick start

To really quick start using PowerToCSS, just add these two line of code inside your `HEAD` tag and you are good to go:

<link rel="stylesheet" href="http://powertoweb.com/powertocss/css/powertocss.css">
<link rel="stylesheet" href="http://powertoweb.com/powertocss/css/standard-theme.css">

Small Footprint

We are trying to build a well architected CSS framework, based on the principles of SMACSS and DRY. We analyze every character of our code!

Easy to Learn

Here, a column name is .column, a button name is .button, a text colored blue is .text-blue. We try to keep things Simple, Easy and Memorable.

Theme able

PowerToCSS provide the base you need to Start and Extend your Project. Sure you can contribute to the community, by adding new themes and new components to the framework.

For You

For you, by people like you! We believe Frameworks should be simple, a starting point for your designs, not the hole design itself. Speed and development Productivity matter!

Things you probably want to know

  1. 1.PowerToCSS is not, nor is trying to be a "Swiss-Knife". With PowerToCSS, we just want to provide a solid CSS base so developers can jump-start any project based in good practices.
  2. 2.PowerToCSS consist in 2 files: the core css file wich is powertocss.css and the theme wich is standard-theme.css. While powertocss.css has the base and the core styles, standard-theme.css has the custom styles
  3. 3.You should never modify the core file wich is powertocss.css
  4. 4.Customizations and modifications are done in standard-theme.css
  5. 5.PowerToCSS is NOT a Rival of the existing solutions, but an Alternative.
